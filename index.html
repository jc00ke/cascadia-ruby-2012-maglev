Simple Example

VM #1

 
  Maglev.abort_transaction
  j = Object.new
  j.object_id
# => 1234567
  Maglev::PERSISTENT_ROOT[:jesse] = j
  Maglev.commit_transaction

VM #2

 
  Maglev.abort_transaction
  Maglev::PERSISTENT_ROOT[:jesse].object_id
# => 1234567

Persistence by Reachability

VM #1

 
Maglev.abort_transaction
names = []
gorby = "Gorby Thunderhorse"
names << gorby
Maglev::PERSISTENT_ROOT[:names] = names
Maglev.commit_transaction
gorby.object_id
# => 9876543

VM #2

 
  Maglev.abort_transaction
  Maglev::PERSISTENT_ROOT[:names].first.object_id
# => 9876543
Those are the same string instances, even though we persisted the names array.
